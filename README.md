## Apresentando o aplicativo
<p>
Aplicativo criado com base na playlist do canal do Hellodev.  
Gerenciador de tarefas (quadro kanban), onde é possível criar, ler, atualizar e excluir tarefas (ToDo / Doing / Done). O aplicativo utiliza o Firebase RealTime Database, além do Firebase Auth.
</p>

## Ferramentas
- Firebase Database
- Firebase Authentication
- Navigation Component
- Safe Args
- Tablayout
- Adapter
- RecyclerView
